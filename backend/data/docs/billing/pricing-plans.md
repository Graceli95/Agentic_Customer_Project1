## Upgrading and Downgrading

### Upgrade Process
1. Select new plan from dashboard
2. Confirm upgrade
3. Pay prorated difference immediately
4. New features available instantly

**Prorated Billing Example:**
- Current plan: Starter ($29/mo)
- Upgrade to Professional ($79/mo) on day 15
- Charge: $79 - ($29 Ã— 15/30) = $64.50

### Downgrade Process
1. Select lower plan from dashboard
2. Downgrade takes effect at end of current billing cycle
3. No immediate charge
4. Receive credit for unused features

**Important:** Downgrading may result in:
- Data over new storage limit (archive or delete required)
- Reduced API limits (requests throttled if over)
- Team member removal (must reduce count)

## FAQ

**Q: Can I change plans anytime?**
A: Yes! Upgrade instantly or downgrade at cycle end.

**Q: What happens if I exceed my limits?**
A: Storage: Uploads blocked until under limit. API: Requests throttled or charged overage ($10 per 10K requests).

**Q: Do you offer refunds?**
A: Yes, 30-day money-back guarantee on first purchase. See Refund Policy for details.

**Q: Is there a setup fee?**
A: No setup fees on any plan!

**Q: Can I pause my subscription?**
A: Not currently, but you can downgrade to Starter plan temporarily.

**Q: What payment date can I choose?**
A: First charge sets your billing date. Contact support to change it.
